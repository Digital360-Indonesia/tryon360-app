# Virtual Try-On Platform

A modern virtual try-on platform that allows users to visualize garments on different models with AI-powered generation.

## Project Structure

This repository contains:
- `try-on-app/` - Main application (frontend + backend)
- `DOCUMENTATION_4_SEPTEMBER_2025_FIXED_FUNDAMENTAL_CHECKPOINT.md` - Latest fixes and improvements

## Quick Start

1. Navigate to the try-on-app directory:
   ```bash
   cd try-on-app
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the application:
   ```bash
   npm start
   ```

## Features

- AI-powered virtual try-on generation
- Multiple model support (Gunawan, Paul, Rachma, Johny)
- Dynamic pose selection
- Embroidery/printing details support
- Local storage for session management
- Real-time generation with debugging capabilities
