# Product Overview

Kustompedia Try-On Platform is an AI-powered garment visualization platform for internal use at Kustompedia. The platform generates high-quality model images for marketing, social media, and web content using multiple AI image generation APIs.

## Core Purpose
Generate consistent model images wearing custom garments with logo enhancements for marketing materials.

## Key Features
- **5-7 Consistent Models**: Pre-defined Indonesian model personas (Johny, Nyoman, Isabella) that maintain exact appearance across generations
- **Multi-AI Support**: OpenAI GPT Image 1, FLUX Pro 1.1, and custom finetuned models
- **Logo Enhancement**: Focus enhancement for sablon, embroidery, and other logo details
- **Queue System**: 5-slot parallel processing for efficient generation management
- **Real-time Monitoring**: Live updates on generation progress and queue status
- **Cost Tracking**: Monitor API costs per generation with quality tiers

## Target Users
Internal Kustompedia team members creating marketing content and product visualizations.

## Business Context
This is an internal tool focused on Indonesian market garment visualization with consistent model personas and branding enhancement capabilities.
